# Gematria (Swift)
Gematria ciphers written in Swift.

## Usage

```swift
var source_string = "Hello, playground"

source_string.ToEnglishOrdinal() // 185
```

## CocoaPods

[CocoaPods](http://cocoapods.org) is a dependency manager for Cocoa projects. You can install it with the following command:

```bash
$ gem install cocoapods
```

> CocoaPods is required to build Gematria 0.0.1+.

To integrate Gematria into your Xcode project using CocoaPods, specify it in your `Podfile`:

```ruby
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '9.0'
use_frameworks!

pod 'Gematria', '~> 0.0.1'
```

Then, run the following command:

```bash
$ pod install
```
